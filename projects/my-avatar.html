Le Projet

My Avatar est un service web inspiré de Gravatar, permettant aux utilisateurs de générer et gérer des photos de profil centralisées.

Développé avec Symfony, ce projet met en avant une architecture robuste reposant sur une API puissante et une base de données MySQL optimisée. L'interface, claire et responsive, a été conçue avec Bootstrap pour garantir une expérience utilisateur fluide sur tous les supports.

Points Techniques

Symfony & API

Création d'endpoints RESTful pour la distribution des avatars et gestion des utilisateurs.

Base de Données

Conception d'un modèle de données relationnel efficace avec MySQL et Doctrine ORM.
